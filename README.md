# Stroop_Test
Stroop_Test for HSS Project

## Only 4 Colours:
### Red , Green , Blue , Yellow

## Input keys:
### 'R' - RED
### 'G'- GREEN
### 'B' - BLUE
### 'Y' - YELLOW

## Total time - 60 seconds

## How to Play:
### Hit the key corresponding to the colour of the text appearing

## Motive:
To measure cognitive abilities of a person
